Rustype - Simple, Focused, and Efficient Markdown Editor

#### Overview

Are you looking for a clean, focused, and efficient Markdown editor? Rustype transforms writing into a simple, enjoyable experience. Perfect for writers, developers, bloggers, and anyone who wants a distraction-free environment for creating Markdown content.

#### Features

- **Native Minimalist Design**: Clean interface that follows macOS design principles, no redundant elements, focus on content
- **Lightning Fast Performance**: Built with SwiftUI for instant startup and smooth response, handles long documents with ease
- **Real-time Preview**: What You See Is What You Get editing experience, write and preview simultaneously in split view
- **Dark Mode Support**: Perfect integration with macOS system dark mode for comfortable writing day or night
- **Full Keyboard Support**: Rich keyboard shortcuts for all operations, boost your writing efficiency
- **Multiple Export Options**: Save as Markdown files or export to HTML format for web publishing
- **Intuitive Toolbar**: Quick access to common Markdown formatting options with one-click buttons
- **File Management**: Easy open, save, and new document operations with familiar macOS file dialogs
- **Markdown Syntax Support**: Full support for headings, bold, italic, code blocks, quotes, lists, tables, links, and more
- **Split View Layout**: Side-by-side editor and preview panels for seamless writing and proofreading
- **Status Bar Information**: Real-time word count, character count, and line count display

#### Data Privacy and Security

- Rustype is an offline-only application that processes all data locally on your device.
- No internet connection is required, and no document data is uploaded to the internet or cloud services.
- We promise not to collect, store, or share any of your documents or personal data.
- All your writings remain confidential and secure on your local device.

#### Is the app native?

Yes, it's native and written in Swift and SwiftUI for optimal performance and seamless macOS integration.

#### Why is this free without ads?

We believe in providing a pure writing experience without distractions. Rustype is designed to help writers and creators produce quality content efficiently, focusing on simplicity and functionality rather than ads or in-app purchases.

#### Can you localize the app into my language?

Currently, Rustype supports English. We are open to adding more languages based on user requests.
